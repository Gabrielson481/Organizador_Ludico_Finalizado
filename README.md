## 📝 Descrição
O **Organizador Lúdico** é um sistema de gestão desktop desenvolvido para otimizar o controle de empréstimos de jogos de tabuleiro. O software foi criado para apoiar o projeto de extensão **Lúdico** da UTFPR (Câmpus Cornélio Procópio), profissionalizando a gestão de recursos utilizados em atividades comunitárias.

O objetivo do sistema é facilitar o acesso da comunidade aos jogos, reduzir o tempo gasto em conferências manuais e eliminar a perda ou extravio de itens do acervo, promovendo o uso eficiente das práticas lúdicas.

## 🚀 Funcionalidades
* **Gestão de Acervo:** Cadastro, edição e exclusão de jogos de tabuleiro, com controle de status (Disponível/Emprestado).
* **Gestão de Usuários:** Cadastro de membros da comunidade autorizados a realizar empréstimos.
* **Controle de Transações:** Registro preciso de empréstimos e devoluções, com histórico vinculado ao usuário.
* **Segurança e Acesso:** Sistema de autenticação com níveis de acesso diferenciados para administradores (membros do projeto) e usuários.
* **Validação de Dados:** Tratamento de exceções robusto, garantindo a integridade das informações (ex: validação de idade e campos obrigatórios).

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **Ambiente de Desenvolvimento:** NetBeans IDE
* **Banco de Dados:** PostgreSQL
* **Interface Gráfica:** Swing (Java)
* **Estrutura de Dados:** ArrayList para gerenciamento dinâmico em memória.
